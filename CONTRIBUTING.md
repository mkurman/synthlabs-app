# Contributing to SynthLabs

First off, thank you for considering contributing to SynthLabs! It's people like you who make this project better for everyone. **All are welcomed** to contribute, regardless of your experience level.

## Ethics & Code of Conduct

We aim to foster an open and welcoming environment. We expect all contributors to:
- Use welcoming and inclusive language.
- Be respectful of differing viewpoints and experiences.
- Gracefully accept constructive criticism.
- Focus on what is best for the community.
- Show empathy towards other community members.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue and include:
- A clear, descriptive title.
- Steps to reproduce the issue.
- Expected vs. actual behavior.
- **Screenshots or videos** showing the bug in action.

### Suggesting Enhancements
If you have an idea for a new feature:
- Open an issue describing the feature.
- Explain why it would be useful.
- If possible, include sketches or mockups of how it might look.

### Pull Requests
We love pull requests! To ensure a smooth review process, please follow these guidelines:

1.  **Fork the repo** and create your branch from `main`. All Pull Requests should be targeted towards the **`dev` branch**.
2.  **Installation**: Run `pnpm install` to install dependencies.
3.  **Style**: Ensure your code follows the existing style (TypeScript + React). Run `pnpm lint` before submitting.
4.  **Descriptive PRs**: Give your PR a meaningful title and description. Explain *what* changed and *why*.
5.  **Visuals**: If your change affects the UI, **you must include a screenshot or GIF** of the change. This helps reviewers understand the impact immediately.
6.  **Update Documentation**: If you've changed how something works, please update the README or relevant docs.

## Local Development

1.  Clone the repository:
    ```bash
    git clone https://github.com/synthlabs/synthlabs-app.git
    cd synthlabs-app
    ```
2.  Install dependencies:
    ```bash
    pnpm install
    ```
3.  Start the development server:
    ```bash
    pnpm dev
    ```

Thank you for being part of the journey!
